# ⚡ Flash-Sale Inventory Order Processor

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![DSA](https://img.shields.io/badge/DSA-Insertion%20Sort%20%26%20Two%20Pointers-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

A Python implementation of a flash-sale inventory processing system. The project demonstrates the use of **In-Place Insertion Sort** and the **Two-Pointer Technique** while maintaining **O(1) auxiliary space** under strict DSA constraints.

## 🚀 Features

- 📦 In-Place Insertion Sort
- 🎯 Two-Pointer Budget Pair Search
- 💾 O(1) Auxiliary Space
- 🧩 Object-Oriented Design
- 🚫 No built-in `sort()`, `sorted()`, dictionaries, sets, or extra arrays

## 📈 Time Complexity

| Operation | Complexity |
|----------|------------|
| Load Orders | O(1) |
| Insertion Sort | O(N²) |
| Two-Pointer Search | O(N) |
